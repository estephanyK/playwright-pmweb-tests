# 🧪 Automação de Testes com Playwright - LetsBook

Este repositório contém uma suíte de testes automatizados E2E utilizando Playwright para validação dos principais fluxos do sistema de reservas e CRM.

---

## 🌐 Aplicação testada

https://lets-teste-qa.lovable.app/

---

## 🎯 Objetivo

Garantir a qualidade dos fluxos críticos da aplicação, validando regras de negócio, integrações de interface e consistência dos dados exibidos ao usuário.

---

## 🧪 Cenários automatizados

- Criação de reserva de hóspede
- Busca de hóspedes no Painel CRM
- Alteração de status de reserva

---

## 🧪 Estratégia de Testes

A estratégia de testes começou pela validação dos principais fluxos críticos da aplicação, iniciando pelo processo de criação de reserva, por ser  parte principal do sistema. Em seguida, foram testados cenários de consulta no Painel CRM e alteração de status de hóspedes, garantindo a integridade do fluxo ponta a ponta.

Foram utilizadas técnicas de testes funcionais E2E e validação de regras de negócio, com foco em fluxos completos do usuário. Também foram aplicadas boas práticas na seleção de elementos para garantir maior estabilidade dos testes.

A priorização dos riscos considerou os fluxos com maior impacto no negócio, como criação de reservas e atualização de status no CRM, pois são funcionalidades essenciais para operação do sistema e diretamente ligadas à experiência do usuário.

A IA foi utilizada como apoio na estruturação dos testes automatizados, na sugestão de boas práticas de automação, na melhoria de seletores e na definição de estratégias de cobertura de testes.
